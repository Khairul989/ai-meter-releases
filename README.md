# AIMeter

A macOS menu bar app for tracking multi-provider LLM usage and quota — Claude, ChatGPT, GitHub Copilot, GLM, Kimi, MiniMax.

## Download

Latest build is in the [Releases](https://github.com/Khairul989/ai-meter-releases/releases) tab.

## Install

1. Download `AIMeter-vX.Y.Z.zip` from the latest release.
2. Unzip and move `AIMeter.app` to `/Applications`.
3. First launch: right-click the app and choose **Open** to clear macOS Gatekeeper.

## Requirements

- macOS 14.0 or later (Apple silicon or Intel).

## Updates

AIMeter checks for updates automatically via [Sparkle](https://sparkle-project.org). New releases appear in-app — no manual download needed after first install.
